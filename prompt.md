# Claude Code 実装依頼

`spec.md` と `impl_plan.md` に従って XFIND を実装してください。

## 作業対象

次を作成または更新してください。

- `src/` 配下のソース一式
- ルートの `CMakeLists.txt`
- `x68k/Makefile`
- `.github/workflows/build.yml`
- `README.md`

## 重要な制約

- 対象は Ubuntu と X68000 / Human68k
- 検索、索引、TUI のコアはなるべく共通化すること
- OS 依存差分は `src/PLATFORM.H` を入口として分離すること
- 外部ライブラリ依存は極力避けること
- 実機 X68000 テストは不要
- Ubuntu での動作確認を優先すること
- ただし Ubuntu 専用実装にしないこと

## 実装の進め方

- `impl_plan.md` のフェーズ順で進めること
- 途中段階でもビルド可能な小さな差分で進めること
- 動作仕様は `spec.md` を正とすること
- 仕様を勝手に拡張しないこと

## 出力してほしいもの

1. 追加・更新したファイル一覧
2. 主要な設計判断の要約
3. ビルド手順
4. 未実装項目がある場合はその明記

## 注意

- `open` と `cd` のみを実装対象とすること
- `edit`、`view`、GUI、全文検索、バイナリインデックスは追加しないこと
- GitHub 公開を前提に、リポジトリとして読みやすい構成にすること
